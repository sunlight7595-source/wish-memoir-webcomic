# 願望備忘錄

《願望備忘錄》第 1 話、第 2 話、第 3 話、第 4 話完整篇 V1。

## 線上發布
這個資料夾已整理成 GitHub Pages 可發布格式：

- `index.html`：第 1 話完整篇
- `episode-02.html`：第 2 話完整篇
- `episode-03.html`：第 3 話完整篇
- `episode-04.html`：第 4 話完整篇
- `episode-01-reader.css`：共用與第 1 話頁面樣式
- `episode-02-reader.css`：第 2 話頁面樣式
- `episode-03-reader.css`：第 3 話頁面樣式
- `episode-04-reader.css`：第 4 話頁面樣式
- `assets/`：漫畫頁圖與預覽圖

## 本機預覽
直接用瀏覽器打開 `index.html`、`episode-02.html`、`episode-03.html` 或 `episode-04.html`。

## GitHub Pages 設定
上傳到 GitHub 後，到 repo 的 `Settings > Pages`，選擇：

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`

儲存後等待 GitHub 產生網址。
