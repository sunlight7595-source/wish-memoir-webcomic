# 願望備忘錄

《願望備忘錄》第 1 話 10 頁試讀完成品 V1。

## 線上發布
這個資料夾已整理成 GitHub Pages 可發布格式：

- `index.html`：作品試讀頁
- `episode-01-reader.css`：頁面樣式
- `assets/`：漫畫頁圖與預覽圖

## 本機預覽
直接用瀏覽器打開 `index.html`。

## GitHub Pages 設定
上傳到 GitHub 後，到 repo 的 `Settings > Pages`，選擇：

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`

儲存後等待 GitHub 產生網址。
